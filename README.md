# 🌍 GeoVista

## 📌 Project Overview

GeoVista is a simple web application that allows users to explore information about countries around the world. It fetches real-time data from the REST Countries API and displays it in a clean and responsive interface.

The project demonstrates the use of JavaScript concepts such as fetching data from APIs, array manipulation, and dynamic DOM updates.

---

## 🚀 Features

* 🔍 **Search Countries**
  Search countries by name using a real-time search input.

* 🌎 **Filter by Region**
  Filter countries based on regions like Asia, Europe, Africa, Americas, and Oceania.

* 📊 **Sort Options**

  * Alphabetical order (A–Z)
  * Population (high to low)
  * Region-based sorting

* 🎚️ **Population Filter**
  Filter countries based on population ranges (small, medium, large).

* ⭐ **Favorites**
  Mark countries as favorites and store them using localStorage.

* 🔄 **Reset Filters**
  Reset all filters and return to the default view.

* 🌗 **Dark / Light Mode**
  Toggle between themes with preference saved in localStorage.

* 📱 **Responsive Design**
  Works smoothly on mobile, tablet, and desktop screens.

## 🛠️ Technologies Used

## 🔗 API Used

* https://restcountries.com/v3.1/all

The API provides country data such as name, flag, capital, population, and region.

  * `.sort()` → sorting data

* Local Storage usage
